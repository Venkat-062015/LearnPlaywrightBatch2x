# VS Code Keyboard Shortcuts for Windows

| Action | Shortcut |
|---|---|
| Command Palette | `Ctrl + Shift + P` |
| Open File | `Ctrl + O` |
| Save File | `Ctrl + S` |
| Save All | `Ctrl + K`, then `S` |
| Close Editor | `Ctrl + F4` |
| Split Editor | `Ctrl + \` |
| Toggle Sidebar | `Ctrl + B` |
| Toggle Terminal | `Ctrl + ` ` (backtick)` |
| New Terminal | `Ctrl + Shift + ` ` (backtick)` |
| Find | `Ctrl + F` |
| Replace | `Ctrl + H` |
| Go to File | `Ctrl + P` |
| Go to Line | `Ctrl + G` |
| Toggle Comment | `Ctrl + /` |
| Format Document | `Shift + Alt + F` |
| Rename Symbol | `F2` |
| Peek Definition | `Alt + F12` |
| Go to Definition | `F12` |
| Show Problems | `Ctrl + Shift + M` |
| Open Settings | `Ctrl + ,` |
| Full Screen | `F11` |
| Zen Mode | `Ctrl + K`, then `Z` |
| Open Extensions | `Ctrl + Shift + X` |
| Open Explorer | `Ctrl + Shift + E` |
